# 🚀 Project Name: YourHeadlineProject

A clean, responsive, and high-performance web project focused on modern UI/UX principles. This project utilizes optimized assets, custom typography, and a modular CSS structure to ensure a seamless user experience.

---

## 📂 Project Structure

The source code is organized as follows:

* **`src/`** — Core development folder.
    * **`css/`** — Layout-specific styles (e.g., `desktop.css`).
    * **`styles/`** — Global styles and design tokens.
    * **`fonts/`** — Custom typography (Oswald, Source Sans Pro).
    * **`img/`** — Optimized visual assets (SVG, PNG, JPG).

---

## ✨ Features

- [x] **Responsive Design:** Fully optimized for desktop and mobile devices.
- [x] **Custom Typography:** Integrated with `Oswald` and `Source Sans Pro` for a unique brand identity.
- [x] **Modular CSS:** Clean and maintainable stylesheets.
- [x] **Performance:** Optimized images and lightweight asset loading.

---

## 🛠️ Technology Stack

* **HTML5** — Semantic structure.
* **CSS3** — Advanced styling with a focus on Flexbox/Grid.
* **Fonts:** Oswald, Source Sans Pro.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### 1. Prerequisites
Make sure you have a modern web browser and a local server (like Live Server for VS Code) or Node.js installed.

### 2. Installation
Clone the repository:
```bash
git clone https://github.com/ErnestSosnovskyi/YourHeadlineProject.git
```

### 3. Usage
Open index.html in your browser or run your development server:

```bash
npm install
npm run dev
```

### 4. Assets Information
This project uses specific graphical assets located in src/img/, including:
```bash
wheel.png & wheel_gray_left.png — Interactive UI components.
```
Optimized background images for high-density displays.
